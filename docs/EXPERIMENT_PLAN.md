# G2 Experiment Plan

## G2-0 Counterfactual Route Audit

Freeze hosts and head. For every sample, execute all admissible hosts.

Metrics:

- selected-host loss;
- oracle admissible-host loss;
- route regret;
- context posterior gap;
- centroid/subspace distance;
- host ablation response;
- conformal coverage and set size;
- calibration inflation;
- compute cost.

## G2-1 Soft admissibility

Add matched-strength penalties for retention, risk, cost, and audit visibility.

## G2-2 Hard admissibility

Project or reject route/lifecycle actions. Report projection magnitude and intransigence.

## G2-3 Transport

Compare Euclidean, learned, and geometry-aware transport with shuffled controls.

## G2-4 Manifold routing

Admit Grassmann/Lie objects only after vector-prototype failure.
